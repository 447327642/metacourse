## railscast课程

### 准备clojure环境

	brew install clojure


### 下载railscast课程

#### 免费课程

	clj railscast-download.clj -rss your-rss-link -type media-format


`media-format`一般填写mp4

### 收费课程

将`your-rss-link`改为订阅后的rss网址。
